# [개념명] 펌웨어의 스테이트머신과 쓰레드 - 상태 기반 흐름 vs 실행 단위

## 1. 스테이트머신
- 상태에 따라 동작을 분기하는 구조
- 메인 루프 안에서 기능을 순차 실행

## 2. 쓰레드
- 독립적인 실행 흐름
- RTOS 또는 OS에서 스케줄링되어 병렬 실행 가능

## 3. 차이
| 항목 | 스테이트머신 | 쓰레드 |
|------|---------------|--------|
| 구조 | 루프 기반 | OS 스케줄링 |
| 실행 | 순차적 | 병렬 또는 선점형 가능 |
| 독립성 | 낮음 | 높음 |

## 4. 결론
- 스테이트머신은 쓰레드처럼 보일 수 있지만 실제 쓰레드는 아님
