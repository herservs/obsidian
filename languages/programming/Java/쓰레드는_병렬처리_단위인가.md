# [개념명] 쓰레드는 병렬처리 단위인가? - 병렬과 동시성의 구분

## 1. 개요
- 쓰레드는 병렬 처리를 위한 논리 단위
- 병렬 실행은 CPU 코어 수에 따라 다름

## 2. 개념 비교

| 개념  | 설명                        |
| --- | ------------------------- |
| 동시성 | 여러 작업을 빠르게 전환             |
| 병렬성 | 여러 작업을 동시에 수행(코어 여러 개 필요) |

## 3. 결론
- 쓰레드는 병렬처리 단위가 아님
- 병렬 처리를 가능하게 하는 기반 구조
